# Quick Start Guide - 3 Steps to Success! 🚀

## Step 1: Download & Extract

Download this project and extract it to a folder.

## Step 2: Connect Your Dosimeter

Plug your Rium GM dosimeter into a USB port.

## Step 3: Launch

### Windows
Double-click: **`START_WINDOWS.bat`**

### Linux / macOS / Raspberry Pi
```bash
chmod +x start.sh
./start.sh
```

### Or use Python directly
```bash
python3 launcher.py
```

## What Happens Next?

The launcher will guide you through:

1. ✅ **Installing dependencies** (if needed)
2. ✅ **Configuring your station** (first time only)
3. ✅ **Testing the connection**
4. ✅ **Starting monitoring**

## That's It!

The system will:
- Automatically detect your dosimeter
- Start recording radiation events
- Save data locally in CSV format
- Optionally upload to OpenRadiation network
