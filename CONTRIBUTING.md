# Contributing

Thank you for your interest in contributing to this ASNR (formerly IRSN) project!

## Quick Guidelines

### Reporting Issues
- Clear description of the problem
- Steps to reproduce
- System information (OS, Python version, hardware)

### Suggesting Features
- Explain the use case
- Consider backward compatibility

### Pull Requests
1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Test thoroughly
5. Submit a pull request

## Code Style
- Follow PEP 8 for Python
- Use English for comments and documentation
- Add docstrings to functions

## Testing
- Test with actual Rium GM hardware if possible
- Verify CSV logging works
- Test API submission (test mode)

## Questions?
Open an issue or discussion on GitHub.
